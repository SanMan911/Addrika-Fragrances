# Quick Image Upload Guide - Addrika Website

## 🎯 What You Need

### 1 Logo + 4 Product Photos

```
📸 LOGO (1 image):
   File: addrika-logo.png
   Size: 200-400px wide
   Format: PNG (transparent background)
   Where: /app/frontend/public/images/logos/

📸 PRODUCTS (4 images):
   File 1: kesar-chandan.jpg
   File 2: regal-rose.jpg  
   File 3: oriental-oudh.jpg
   File 4: bold-bakhoor.jpg
   Size: 800-1200px wide each
   Format: JPG or PNG
   Where: /app/frontend/public/images/products/
```

---

## 🚀 3 Ways to Upload

### METHOD 1: Emergent File Browser ⭐ (Easiest)

1. Look for "Files" or "Explorer" icon on left sidebar
2. Navigate: frontend → public → images
3. Right-click "logos" folder → Upload Files
4. Upload your logo as "addrika-logo.png"
5. Right-click "products" folder → Upload Files  
6. Upload 4 photos with exact names above

### METHOD 2: Send to Me 💬 (Super Easy)

Share your images via:
- Google Drive link
- Dropbox link  
- Any image hosting URL
- Email attachment

I'll upload them for you!

### METHOD 3: GitHub (If Connected)

1. Clone repository
2. Add images to folders
3. Push to GitHub
4. Changes deploy automatically

---

## ⚠️ IMPORTANT: File Naming

**Must use EXACT filenames** (lowercase, hyphens):

✅ CORRECT:
- addrika-logo.png
- kesar-chandan.jpg
- regal-rose.jpg
- oriental-oudh.jpg
- bold-bakhoor.jpg

❌ WRONG:
- Addrika Logo.png (spaces, caps)
- Kesar_Chandan.JPG (underscore, caps)
- product1.jpg (generic name)
